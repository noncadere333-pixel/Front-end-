# Лаба 2

## Цель работы
Изучить работу с React-компонентами, фрагментами, передачей данных через props и рендерингом списков с помощью метода map.


## Структура проекта

Lab_02 состоит из двух заданий:

- **task1** — работа с React Fragment и отображение списков
- **task2** — создание переиспользуемых компонентов и вложенных структур


## Task 1
В первом задании реализованы:
- использование React Fragment без лишних div
- рендеринг списка с помощью метода map
- использование свойства key
- вывод количества элементов списка


## Task 2
Во втором задании реализованы:
- компонент Card с поддержкой props и children
- компонент Section с заголовком
- отображение списка продуктов
- переиспользуемые компоненты


## Запуск проекта

Для запуска любого задания выполните команды:

npm install
npm run dev
