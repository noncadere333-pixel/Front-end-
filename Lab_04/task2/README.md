В данном проекте выполнен рефакторинг монолитного компонента управления статьями
в набор переиспользуемых React-компонентов.

## Структура компонентов
- ArticleManager — управляет состоянием статей и логикой приложения
- AddArticle — отвечает за добавление новых статей
- ArticleList — отображает список статей
- ArticleItem — отображает одну статью и управляет локальным состоянием раскрытия

## Реализованные принципы
- Разделение ответственности компонентов
- Передача данных и обработчиков через props
- Использование локального состояния (useState) в дочерних компонентах
- Повышение читаемости и масштабируемости кода

## Используемые технологии
- React (function components)
- Хук useState
- Vite

## Запуск проекта
1. Установить зависимости:
   npm install
2. Запустить проект:
   npm run dev

Проект соответствует требованиям лабораторной работы 4.2.
