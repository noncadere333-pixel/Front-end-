# Лабораторная работа 05 — TypeScript и проверка props

## Задание 1
Создание типизированных React-компонентов с использованием интерфейсов (interfaces) и псевдонимов типов (type aliases) в TypeScript.

## Задание 2
Приложение для поиска пользователей с использованием типизированного состояния (state) и обработчиков событий.

## Основные изученные концепции
- интерфейсы (interfaces)
- объединённые типы (union types)
- типизированные props
- типизированный useState
- React.ChangeEvent для обработки событий
