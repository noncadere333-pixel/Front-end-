# Lab_06 - React Router и навигация

Этот проект показывает работу маршутизации в React с использованием библиотеки react-router-dom.

## Возможности приложения 
- Многостраничная навигация
- Поддержка TypeScript
- Query-параметры

## Маршруты приложения
/-> Главная страница
/courses -> Список курсов
/courses/:id -> Страница конкретного курса 
/about -> Страница с информацией о программе
Неизвестные маршруты -> страница 404
