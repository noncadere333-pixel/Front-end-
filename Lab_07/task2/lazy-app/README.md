# Lab 07 — Разделение кода (Code Splitting) с использованием Suspense

В этом проекте демонстрируется использование разделения кода (code splitting) в React.

## Используемые технологии

React  
TypeScript  
React Router  

## Реализованные концепции

### Code Splitting
Разделение кода позволяет разбивать приложение на небольшие части, которые загружаются только тогда, когда они действительно нужны. Это ускоряет первоначальную загрузку приложения.

### React.lazy()
Функция React.lazy() используется для динамического импорта компонентов. Благодаря этому страницы приложения загружаются только при переходе на соответствующий маршрут.

### Suspense
Компонент Suspense отображает запасной интерфейс (fallback UI), пока лениво загружаемый компонент ещё не загрузился.

### Error Boundary
Error Boundary перехватывает ошибки в дочерних компонентах и отображает пользовательский интерфейс с сообщением об ошибке, не позволяя приложению полностью сломаться.
